Goal: Develop a web application for my profile.

How should it look:

Typical Header: Name, college, Degree, small photo, bio to linkedin, contact etc.

Main content: 

1. It should be a timeline from when I started my career to current time
2. The content of the file should be taken from my current resume.
    1. But I should be able to easily add projects at a particular timeline
3. The project content should be a brief overview of the project (initially from my resume), technologies and algorithms used
4. The timelines should flow very smoothly and should be clickable. For example I can expand on a particular year or company to see what has been done during that time.
5. The final drill down would be the project level details
6. This is not exhaustive and would evolve. We will also update this readme based on the new requirement

Development
